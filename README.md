# Practice-platformer-

## Описание игры
Cyber Rooftop Adventure Ultimate — это динамичный 2D платформер с киберпанк-эстетикой, где игроку предстоит преодолевать опасные крыши небоскребов, собирать монеты и сражаться с врагами. Игра сочетает классический геймплей платформеров с современными визуальными эффектами и процедурной генерацией уровней.

## Особенности игры:
🏙️ Процедурная генерация уровней — каждый уровень уникален

🤖 Различные типы врагов с ИИ-поведением

💰 Система сбора монет для подсчета очков

⚔️ Три уровня сложности: Easy, Normal, Hard

🎵 Динамическое музыкальное сопровождение

✨ Стилизованная киберпанк-графика

💫 Система неуязвимости после получения урона

## Сюжет:
Вы - кибер-бегун, путешествующий по крышам футуристического мегаполиса. Ваша цель — добраться как можно дальше, собирая ценные ресурсы и избегая опасностей, которые поджидают на каждом шагу.

# Как запустить игру?

1) Установить Python (3.12.5)
2) Установить Pygame

```pip install pygame ```

3) Скачать файл game.v1.0.py
4) Запустить файл

``` python game.v1.0.py ```


# Игровая механика:
Прыжок на врагов сверху позволяет уничтожать их

Шипы наносят урон при касании

Падение с платформы отнимает жизнь и возвращает на последнюю безопасную позицию

Сбор монет увеличивает счет

Полоса здоровья отображается в верхней части экрана

# Использованные технологии
Библиотеки:
Технология	Версия	Назначение
Pygame	2.x	Основной игровой движок: графика, звук, обработка ввода, спрайты
NumPy	1.x	Генерация звуковых волн и математические расчеты

#Технические особенности реализации:
1. Графический движок
Рисование спрайтов через pygame.Surface с поддержкой альфа-канала

Процедурная генерация персонажей и врагов

Система параллакса для фона (звезды, здания)

Камера, следующая за игроком

2. Звуковая система
Генерация звуков через синтез синусоидальных волн (NumPy)

Динамическое музыкальное сопровождение из сгенерированных нот

Регулировка громкости в реальном времени

3. Физический движок
Реализация гравитации и ускорения

Система коллизий с разделением по осям X и Y

Трение для плавного движения

4. Искусственный интеллект
Патрулирование врагов с заданным радиусом

Проверка опоры для смены направления

Реакция на игрока при прыжке

5. Геймдизайн
Процедурная генерация уровней с нарастающей сложностью

Система уровней сложности, влияющая на скорость врагов и урон

Балансировка здоровья и урона
