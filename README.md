# Practice-platformer-

## Описание игры
Cyber Rooftop Adventure Ultimate — это динамичный 2D платформер с киберпанк-эстетикой, где игроку предстоит преодолевать опасные крыши небоскребов, собирать монеты и сражаться с врагами. Игра сочетает классический геймплей платформеров с современными визуальными эффектами и процедурной генерацией уровней.

## Особенности игры:
🏙️ Процедурная генерация уровней — каждый уровень уникален

🤖 Различные типы врагов с ИИ-поведением

💰 Система сбора монет для подсчета очков

⚔️ Три уровня сложности: Easy, Normal, Hard

🎵 Динамическое музыкальное сопровождение

✨ Стилизованная киберпанк-графика

💫 Система неуязвимости после получения урона

## Сюжет:
Вы - кибер-бегун, путешествующий по крышам футуристического мегаполиса. Ваша цель — добраться как можно дальше, собирая ценные ресурсы и избегая опасностей, которые поджидают на каждом шагу.

# Как запустить игру?

1) Установить Python (3.12.5)
2) Установить Pygame

```pip install pygame ```

3) Скачать файл game.v1.0.py
4) Запустить файл

``` python game.v1.0.py ```


# Использованные технологии (библиотеки):
1) Pygame	2.x	Основной игровой движок: графика, звук, обработка ввода, спрайты.
2) NumPy	1.x	Генерация звуковых волн и математические расчеты.

## Игровая механика:
1) Прыжок на врагов сверху позволяет уничтожать их;
2) Шипы наносят урон при касании;
3) Падение с платформы отнимает жизнь и возвращает на последнюю безопасную позицию;
4) Сбор монет увеличивает счет;
5) Полоса здоровья отображается в верхней части экрана.

## Технические особенности реализации:
1. Рисование спрайтов через pygame.Surface.

2. Генерация звуков через синтез синусоидальных волн (NumPy).
   
4. Реализация гравитации и ускорения. Система коллизий с разделением по осям X и Y.

5. Патрулирование врагов с заданным радиусом.

6. Процедурная генерация уровней с нарастающей сложностью.
